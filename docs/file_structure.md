# File structure

```
assets/  -- non-code files
docs/  -- documentation
src/  -- all code
  engine/  -- common non-portable functionality
  entities/  -- entity factories
  lib/  -- common portable functionality
  systems/  -- systems
  init_ecs.py  -- initialization of the game itself
fallen.py  -- entrypoint
Makefile  -- development scripts
README.md
requirements.txt  -- python user-side requirements
```
